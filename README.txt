Lush Scented Paradise 芬芳茶园

🌸 Lush Scented Paradise Mod 芬芳茶园模组🌿
一个以鲜花、茶叶、农作物和饮品为主题的 Minecraft 模组，添加了丰富的饮品系统，让玩家可以在游戏中制作和享受各种特色饮品。

Lush Scented ParadiseMod 旨在为 Minecraft 世界增添更多生活气息和趣味性。
模组新增了 七十余种独特饮品，包括 花卉茶、果汁、茶、调制饮品 等，让玩家可以通过种植、采摘、晒干、煮茶等方式制作美味饮品，为探险和战斗增添一份悠闲的乐趣。

✨ 模组特色
🌿 丰富的植物与作物

新增 多种可种植农作物 和 可采集鲜花，作为饮品的主要原料。

🍵 七十余种饮品

茶饮：绿茶、红茶、乌龙茶、普洱茶、薄荷茶等。
花茶：玫瑰花茶、蒲公英花茶、茉莉花茶等。
特色饮品：蜂蜜薄荷水还有更多奇妙的饮品等你来探索。

🍶独特的酿造与调制系统
使用 全新晾晒和煮茶方块 进行晾晒 泡茶、冲泡果茶、制作花茶 等，真实还原饮品制作过程。
不同饮品提供不同的效果，玩家可以在游戏中自行探索。

🏡 悠闲的Minecraft生活

采摘鲜花 & 种植作物，打造自己的茶园或果园。
饮品调配，尝试各种配方，制作独特的饮品！

🔧 安装指南
##前置需求
Minecraft 版本：1.20.1
Forge 47.3.0+（建议使用最新版本的forge（模组构建时的47.3.33））

##安装步骤
下载本模组 （https://legacy.curseforge.com/minecraft/mc-mods/lush-scented-paradise）
将 .jar 文件放入 mods 文件夹（路径：.minecraft/mods）
运行 Minecraft 并选择对应的 Forge 版本
享受你的饮品之旅！

# 🌸 Lush Scented Paradise Mod 🌿
*A Blooming Tea Garden Experience for Minecraft*

![Banner](docs/banner.png) <!-- Add your mod banner image here -->

## 🍵 About This Mod
**Lush Scented Paradise** brings vibrant floral life and sophisticated beverage crafting to Minecraft! This mod adds over **70 unique drinks** including teas, floral infusions, and magical brews, transforming your survival experience into a serene gardening adventure.


## ✨ Key Features
### 🌸 **Botanical Wonders**
- 12+ new **cultivable crops** and **wild flowers**
- Interactive gardening system with cross-breeding mechanics
- Seasonal bloom cycles for enhanced realism

### 🫖 **70+ Beverage Recipes**
| Category       | Examples                          | Special Effects                  |
|----------------|-----------------------------------|-----------------------------------|
| **Classic Tea**| Green Tea, Oolong, Pu'erh        | Regeneration, Night Vision       |
| **Floral Tea** | Rosehip, Jasmine, Chrysanthemum  | Speed Boost, Poison Resistance   |
| **Magic Brews**| Starlight Infusion, Netherbloom  | Fire Resistance, Levitation      |

### ☕ **Artisanal Crafting**
- **New Functional Blocks**:
  - ☀️ **Drying Rack** - Sun-cure tea leaves and flowers
  - 🍶 **Tea Brewing Station** - Precise temperature control system
  - 🧪 **Infusion Cauldron** - Mix magical ingredients
- **Dynamic Brewing System**:
  - Water quality matters (rainwater vs. spring water)
  - Fermentation mechanics for aged teas
  - Customizable drink packaging

## 📦 Installation
### Requirements
- **Minecraft** 1.20.1
- **Forge** 47.3.33+ ([Download](https://files.minecraftforge.net/))

### Quick Setup
```bash
1. Download latest .jar from [CurseForge](https://legacy.curseforge.com/minecraft/mc-mods/lush-scented-paradise)
2. Place in .minecraft/mods folder
3. Launch Minecraft with Forge profile

## Clone GITHUB
1. git clone https://github.com/roylinglrl/LushScentedParadise.git
2. ./gradlew build
